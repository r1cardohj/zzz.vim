# zzz.vim

it is a soft, simple color scheme for Vim, inspired by the colors of the [Smyck](https://github.com/hukl/Smyck-Color-Scheme)

## Feature

### Lang support

#### python

* f-string syntax highlight
* cls and self syntax highlight
* @decorator highlight

#### other Language

* coming soon...

### Plug Support

* ale
* nerdtree
* nerdtree-git-plugin
* coc
* vim-matchup
* ...

## Installation

```vim
Plug 'r1cardohj/zzz.vim'
```
